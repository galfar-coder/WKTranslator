# WKTranslator

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

A localization and translation framework for **White Knuckle**.

## Project Overview
**WKTranslator** is a community-driven tool designed to bridge the gap between the game's current state and its future official localization. This mod allows players and community translators to implement custom language support immediately.

> **[IMPORTANT]**
>
> This mod is intended as a temporary/community solution until official localization is implemented by the developers in the base game.

## Features
* **Dynamic Hooking:** Targets in-game text fields for real-time translation.
* **Open Framework:** Built to be easily modified and extended by the community.
* **Lightweight:** Designed to run with minimal overhead using BepInEx.

## Current Status
This mod is a work in progress. As it is a community tool meant to precede official updates, please expect:
* Some UI elements may remain in the default language.
* Potential formatting adjustments as the game's official localization draws nearer.

## Getting Started
### Prerequisites
* [BepInEx 5.x](https://github.com/BepInEx/BepInEx)
* [White Knuckle](https://store.steampowered.com/app/3195790/White_Knuckle/)

### Installation

#### Using a mod manager
1. Get/Open your mod manager of choice *(r2modman, Gale, etc.)*
2. Go to the Online/Get Mods tab
3. Search for `WKTranslator` and download it
4. Launch the game to generate the necessary configuration files.

#### Manual
1. Download the latest release.
2. Place the `WKTranslator.dll` into your `White Knuckle/BepInEx/plugins` folder.
3. Launch the game to generate the necessary configuration files.
